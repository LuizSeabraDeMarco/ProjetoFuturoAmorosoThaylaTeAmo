APOSTILA – QUÍMICA ORGÂNICA

1. Como identificar cada função orgânica

Aqui estão formas de identificação + exemplos:

  • Hidrocarbonetos → só C e H. Exemplo: Propano → CH3–CH2–CH3
  • Alcanos → ligações simples. Exemplo: Metano → CH4
  • Alcenos → dupla ligação. Exemplo: Eteno → CH2=CH2
  • Alcinos → tripla ligação. Exemplo: Etino → CH≡CH
  • Álcool → –OH ligado a C saturado. Exemplo: Etanol → CH3–CH2–OH
  • Fenol → –OH no benzeno. Exemplo: Fenol → C6H5–OH
  • Aldeído → –CHO na ponta. Exemplo: Etanal → CH3–CHO
  • Cetona → C=O entre carbonos. Exemplo: Acetona → CH3–CO–CH3
  • Ácido carboxílico → –COOH. Exemplo: Ácido acético → CH3–COOH
  • Éster → –COO–. Exemplo: Acetato de etila → CH3–COO–CH2–CH3
  • Éter → C–O–C. Exemplo: Etil metil éter → CH3–O–CH2–CH3
  • Amina → –NH2. Exemplo: Metilamina → CH3–NH2
  • Amida → –CONH2. Exemplo: Etanamida → CH3–CONH2

2. Exemplos estruturais das funções

Alguns exemplos adicionais:

Benzeno (aromático): C6H6

Butilamina (amina): CH3–CH2–CH2–CH2–NH2

Butanal (aldeído): CH3–CH2–CH2–CHO

Dietil éter (éter): CH3–CH2–O–CH2–CH3

Propionato de metila (éster): CH3–CH2–COO–CH3

3. Onde cada função aparece na sociedade

  • Álcool: Etanol no combustível.
  • Aldeído: Formaldeído em resinas e MDF.
  • Cetona: Acetona usada como solvente.
  • Éster: Aromas artificiais (ex.: banana).
  • Amida: Nylon usado em roupas.
  • Ácido carboxílico: Vinagre (ácido acético).
  • Amina: Adrenalina contém grupo amina.

4. Nomenclatura das principais ramificações (substituintes)

Ramificações são grupos ligados à cadeia principal e aparecem antes do nome do composto. Elas normalmente terminam em –il.

  • Metil (–CH3) — Ex.: 2-metilpropano
  • Etil (–CH2–CH3) — Ex.: 3-etilhexano
  • Propil (–CH2–CH2–CH3) — Ex.: 4-propilheptano
  • Isopropil ((CH3)2CH–) — Ex.: 2-isopropilbutano
  • Butil (–CH2–CH2–CH2–CH3) — Ex.: 5-butiloctano
  • Sec-butil — Ex.: 3-sec-butilpentano
  • Ter-butil ((CH3)3C–) — Ex.: 2-ter-butilpropano
  • Isobutil ((CH3)2CH–CH2–) — Ex.: 4-isobutilnonano
  • Halogênios: fluoro-, cloro-, bromo-, iodo → Ex.: 1-bromopropano
  • Ramificações repetidas: di-, tri-, tetra- → Ex.: 2,3-dimetilbutano

5. Nomenclatura IUPAC + exemplos

  • 2-metilbutano → cadeia principal de 4 carbonos + metil no C2
  • 3-hexeno → dupla no carbono 3
  • Ácido butanoico → 4 carbonos + COOH
  • Propanona → 3 carbonos
  • Metoxietano → CH3–O–CH2–CH3

6. Hibridização dos carbonos + exemplos

  • sp³ → alcanos (Ex.: CH4)
  • sp² → alcenos e aromáticos (Ex.: CH2=CH2)
  • sp → alcinos (Ex.: CH≡CH)

7. O que é carbono quiral + exemplo

Carbono quiral: ligado a 4 grupos diferentes.

Exemplo: 2-butanol → C2 é quiral.

8. Classificação das cadeias carbônicas + exemplos

  • Aberta saturada: CH3–CH2–CH2–CH3
  • Aberta insaturada: CH2=CH–CH3
  • Aromática: C6H6
  • Fechada heterogênea: Piridina
  • Ramificada: 2-metilpropano

9. Isomeria espacial (geométrica e óptica) + exemplos

  • Isomeria geométrica (cis/trans):
    Ácido maleico (cis) e ácido fumárico (trans)
  • Isomeria óptica:
    Exemplo: 2-butanol → possui enantiômeros.

10. Exemplos de isomeria na medicina

  • Talidomida → um isômero é terapêutico; outro é tóxico.
  • Ibuprofeno → isômero S é mais ativo.
  • Adrenalina → apenas um enantiômero é usado.
  • L-Dopa → usada no tratamento do Parkinson.
